## ✦ What is YallBot?

YallBot is a slick, developer-focused AI chatbot that runs entirely in your browser. It's powered by the **Claude API** (claude-sonnet-4) and styled to match the YallCode aesthetic — dark, clean, code-terminal vibes.

No backend. No server. Just paste your API key and start chatting.

---

## ✦ Features

- 🤖 Powered by **Claude** (claude-sonnet-4) via the Anthropic API
- 🔑 API key entered in-browser — never stored or shared
- 💬 Full conversation memory within the session
- 🎨 Markdown rendering — code blocks, bold, lists, headers
- ⚡ Typing indicator + live status dot
- 🧹 One-click chat clear
- 💡 Quick-start suggestion chips
- 📱 Responsive — works on mobile too

---

## ✦ How to Use

**1. Get a free API key**

Go to [console.anthropic.com](https://console.anthropic.com) → create an account → generate an API key (starts with `sk-ant-`).

> Anthropic offers free credits to new accounts.

**2. Open YallBot**

👉 [yallcode.github.io/YallBot](https://yallcode.github.io/YallBot/)

**3. Paste your key & chat**

Enter your key in the modal → click **Launch YallBot** → start asking anything!

## ✦ Run Locally

```bash
git clone https://github.com/yallcode/YallBot.git
cd YallBot
# Open index.html in your browser — that's it!
open index.html
```

No dependencies. No build step. Pure HTML/CSS/JS.
